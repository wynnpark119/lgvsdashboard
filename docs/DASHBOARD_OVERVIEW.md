# LG VS MTK Dashboard Overview

## 대시보드 목적

OEM의 LG VS 기술에 대한 **관심 상태**를 파악하고,  
"이해를 동반한 접촉이 발생한 기술"을 식별하여  
기술 조직과의 협업 시점을 판단하기 위한 대시보드

---

## 대시보드 전체 구조 (Page Architecture)

### 1. 전체 요약 (Overview)
**경로:** `/`

- 전체 기술 관심 흐름 요약
- B2B 퍼널 현황 (TOFU / MOFU / BOFU)
- 기술별 관심 신호 및 추세
- 캠페인/이벤트 영향 요약

### 2. 기술별 관심 현황 (Progress)
**경로:** `/review-flow/*`

| 하위 페이지 | 경로 | 설명 |
|------------|------|------|
| 전체 현황 | `/review-flow` | 기술별 관심 분포 및 추이 |
| 처음 접촉 (TOFU) | `/review-flow/tofu` | 최초 유입, 인지 형성 여부 판단 |
| 깊은 관심 (MOFU/BOFU) | `/review-flow/mofu-bofu` | 반복 접촉, 심화 콘텐츠 소비, 문의 발생 |
| 기술별 비교 | `/review-flow/technology` | 기술 간 관심도 비교 |
| 추세 | `/review-flow/momentum` | 시간에 따른 관심 변화 트렌드 |

### 3. 캠페인/광고 영향 (Campaign)
**경로:** `/campaign-impact/*`

| 하위 페이지 | 경로 | 설명 |
|------------|------|------|
| 영향 분석 | `/campaign-impact` | 캠페인 전·후 관심 단계 이동 분석 |
| 광고 영향 | `/campaign-impact/paid` | Paid Media가 관심 흐름에 미친 영향 |
| 콘텐츠 영향 | `/campaign-impact/content` | 콘텐츠별 소비 패턴 및 기여도 |
| 캠페인 목록 | `/campaign` | 전체 캠페인 리스트 및 YoY 비교 |

### 4. 상세 데이터 (Detail)
**경로:** `/detail/*`

| 하위 페이지 | 경로 | 설명 |
|------------|------|------|
| 기술별 | `/detail/technology` | 기술별 방문 히트맵, 상세 지표 |
| 채널별 | `/detail/channel` | 채널별 역할 (Converter/Attractor/Igniter) |
| 문의 & 뉴스레터 | `/detail/inquiry` | Inquiry 현황, 뉴스레터 구독자 추이 |
| 콘텐츠 | `/detail/content` | 콘텐츠별 조회, 완료율, OEM 관심 |

## B2B 퍼널 정의

| 단계 | 정의 | 주요 신호 |
|------|------|----------|
| **TOFU** (처음 접촉) | 최초 유입, 인지 형성 | 첫 방문, 단일 콘텐츠 조회 |
| **MOFU** (반복 접촉) | 관심 지속, 심화 탐색 | 재방문, 복수 콘텐츠 소비, 웨비나 등록 |
| **BOFU** (깊은 관심) | 구체적 관심 표현 | Inquiry 발생, 스펙 요청, 자료 다운로드 |

## 대표 성과 지표

### 핵심 지표 (Primary)
- **TOFU → MOFU 전환율**: 재방문 발생 비율
- **MOFU → BOFU 전환율**: Inquiry/스펙 요청 발생 비율
- **기술별 관심 추세**: 30/60/90일 관심도 변화
- **콘텐츠 완료율**: 끝까지 소비한 비율 (70%+ = 높은 관심)
- **뉴스레터 구독자 순증**: 월간 신규 - 해지

### 보조 지표 (Secondary)
- 페이지 조회 수
- 클릭 수
- 노출 수

※ 단순 노출·클릭 수는 보조 지표로만 활용

## 대시보드 출력 관점

### 정렬 기준
- 기술/솔루션 단위로 정렬
- "많이 본 기술"이 아니라 **"이해를 동반한 접촉이 발생한 기술"** 강조

### 판단 기준
| 신호 | 의미 |
|------|------|
| 체류 시간 3분+ | 콘텐츠 깊이 있게 소비 |
| 영상 50%+ 시청 | 기술에 대한 실질적 관심 |
| 복수 접점 발생 | 단일 기술에 대한 지속적 관심 |
| Inquiry/스펙 요청 | 구체적 검토 의사 표현 |
| 뉴스레터 구독 | 지속적 관심 유지 의향 |

### 광고(Paid Media)의 역할
- 독립 KPI로 평가하지 않음
- **"관심 흐름을 얼마나 증폭/가속했는가"**로 해석
- 광고 종료 후에도 관심이 유지되는지 확인

## 이 대시보드의 역할

1. **기술 관심의 현재 상태 파악**
   - 어떤 기술이 주목받고 있는가
   - 관심이 증가/유지/하락 중인가

2. **관심 단계 식별**
   - 처음 접촉 단계인가
   - 깊은 관심 단계로 이동했는가

3. **기술 조직 협업 시점 판단**
   - BOFU 단계 진입 시 기술 조직에 현황 공유
   - 콘텐츠 고도화 방향 도출

4. **캠페인 효과 해석**
   - 캠페인이 관심 흐름에 어떤 영향을 미쳤는가
   - Amplifier / Accelerator / Noise 분류

## 대시보드에서 하지 않는 것

| 항목 | 이유 |
|------|------|
| CTR/CPC/ROI 평가 | 성과 마케팅 KPI는 별도 도구에서 관리 |
| 영업 리드 전달 | LG VS 마케팅팀의 역할 범위 아님 |
| 자동 인사이트/추천 | 판단은 사용자 몫, 데이터만 제공 |
| 채널 간 효율 비교 | 채널별 역할이 다르므로 비교 지양 |
